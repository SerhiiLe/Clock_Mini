about this clock
System Information

A clock based on an LED matrix with the MAX7219 chip (github). Created based on my addressable LED project.

Clock features:

  • Display the time :)
  • Adjust time using buttons without the internet.
  • Multiple alarms that can be set for one-time, repeating, or specific weekdays.
  • Different beep options for each alarm.
  • A ticker message display. Several messages are supported, each with configurable repeat mode.
  • Automatic matrix brightness control based on ambient light (so it does not disturb at night).
  • Ability to change matrix brightness by time, including sunrise/sunset.

You can send a message to the display through the web interface in this format:

  • http://hostname/show?msg=MESSAGE_TO_SEND - show the message once.
  • http://hostname/show?msg=MESSAGE_TO_SEND&cnt=XX - show the message XX times every 30 seconds.
  • http://hostname/show?msg=MESSAGE_TO_SEND&cnt=XX&int=YY - show the message XX times with an interval of YY seconds.
  • http://hostname/who - returns the clock name.
You can shorten msg, cnt, int to m, c, i.

Almost all clock control is done through the web interface because it is difficult to manage many options with only a couple of buttons. There are two buttons and a motion sensor for control. The motion sensor is used to turn off the alarm, which makes sense: if someone is moving near the clock, they have likely woken up. Button actions depend on the number of presses:

Button 1 (Select)

  1. Show temperature and pressure, turn off the alarm.
  2. Show weather from the internet.
  3. Show a quote.
  4. Update weather from the internet.
  5. Update the quote.
Long press enters time setting mode. Short presses after that select hours, minutes, day, month, year. Another long press exits the mode.

Button 2 (Set)

  1. Show the date in long format, turn off the alarm.
  2. Show help for the second button (Select).
  3. Show the current IP address to connect to the clock.
  4. Show the current light sensor values.
  5. Reset the web password or WiFi settings.
Long press shows help. It is contextual and depends on the current mode.
